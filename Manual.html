  1. 일반 : 백색선
  2. 임시 : 황색선
  3. 특별 : 터널, 관리동 옆, 118-¾ 라인
  4. 불법 : 임시 - 주차 가능 시간 (18:00 ~ 익일 10:00) 이외, 공휴일은 제외
    5.           특별 - 주차 가능 시간 (22:00 ~ 익일 10:00) 이외, 요일과 무관
  5. 24시간 단속 대상
    6. - 램프 구간 (111-¾ 라인)
    - 중앙 도로 (분홍색)
    - D46, D47 주변 (3G에서 들어가면)
    - 현관문 앞
    - 106-¾ 라인 상가 출입문, C15
    - 불법 주차
    - 이중 주차
    - 주차선 위반
    - 회차 구역
  6. 주차 단속 요령
    - 가장 자리를 따라 이동 하면서 한다.
    - 안쪽 부분에 단속 대상이 있을 때는 단속 후 가장 자리로 이동한다.

3G 근무자 방문차량 응대 매뉴얼

  1. 어디 가십니까?
        동호수를 입력한다
  2. 전화번호를 말씀해 주십시오.
        전화번호를 입력한다.
  3. 무슨 용무로 가십니까?
        배달이면 배달을 선택 클릭하고 확인을 클릭한다.
  4. 방문증 뽑아서 가십시오
        -(배송차량일 때) 비상등켜고 서행하십시오.
        - 지상 게이트 열림 키를 조작한다.

♣ 당근거래, 중고거래, 금융관련, 서류 관련, 부동산 관련 차량은 세대 인터폰으로 확인 후 입차 시킨다.
