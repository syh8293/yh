특별한 성경 어휘 : "tion" (명사형 어미)

  1. 『구원』(Salvation) : 죄와 지옥으로부터 구원받는 전체적인 작용.
  2. 『구속』(Redemption) : 무언가를 다시 사는 것.
  3. 『의롭게 하심』(Justification) : 의롭다고 선언하는 것.
  4. 『거룩케 하심』(Sanctification) : 따로 구별하는 것 또는 씻는 것.
  5. 『입양』(Adoption) : 죄인을 하나님의 가족으로 인정하는 법적인 행위.
  6. ‘전가’(Imputation) : 우리의 죄들에 대한 책임을 예수 그리스도께 부가시키고, 그리스도의 의를 우리에게 부가시키는 법적인 행위.
  7. 『화목제물』(Propitiation) : 무언가를 만족시키기 위해 지불되는 댓가.
  8. 『중생』(Regeneration) : 죄인의 영이 다시 태어나게 하는 성령의 역사.
  9. 『사하심』(Remission) : 용서.
  10. 『할례』(Circumcision) : 구원받은 혼과 영으로부터 죄 있는 육신의 몸을 베어 내는 성령의 역사.
  11. 『화해』(Reconciliation) : 장애물을 제거하는 것.
  12. “영광스럽게 됨”(Glorification) : 죄 없는 상태에 도달하는 것.

현대의 위조된 “성경들”을 검토할 때는, 이상의 단어들이 얼마나 많이 어떻게 삭제되고 바뀌었는지를 점검하고 확인해야 한다. 특히 『교리』 (doctrine)란 단어를 제거하기 위해 “교훈” (teaching)이란 단어로 대체한 것에 주의하라. 『종들』 (servants)과 『종』 (servant)이란 단어를 어떻게 “노예들” (slaves)과 “노예” (slave)란 단어로 대체시켰는지 주목하라. 『악한』 (evil)이란 단어를 얼마나 여러 번 완전히 가려 버렸는지 세어 보고, 얼마나 여러 번 『음행』 (fornication)이란 단어를 던져 버리고 “부도덕” (immorality)이란 부드러운 단어를 삽입시켰는지 헤아려 보라. “좋은게 좋은 것이고, 지옥은 어디에도 없다.”는 식이다. - 적어도 마태복음 11:23; 16:18, 누가복음 10:15; 16:23, 사도행전 2:27, 31, 요한계시록 1:18; 6:8; 20:13-14의 경우에서는 그렇다.

그것이 바로 1880년에서 현재까지의 모든 현대 역본들 배후에 있는 “경건한” 학자들의 관념인 것이다. 그러한 것들은 헤롯이나 이세벨만큼이나 “경건”하다.
