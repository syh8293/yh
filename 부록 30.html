재림까지의 안식년 주기

환희의 해 (Jubilee)는 매 49년 후에 지켜야만 했다 (레 25:8-9). 하나님은 역사 속에서 열 번의 '환희의 해'의 주기, 곧 490년의 단위로 일하신다 (단 9:24).

1) B.C. 4004년을 창조로 잡고 열 번의 환희의 해를 게산하면서 내려가면 다음과 같다.

  1. B.C. 4004년 + 10번의 환희의 해 (490년) = B.C. 3514년
  2. B.C. 3514년 + 10번의 환희의 해 = B.C. 3024년
  3. B.C. 3024년 + 10번의 환희의 해 = B.C. 2534년
  4. B.C. 2534년 + 10번의 환희의 해 = B.C. 2044년
  5. B.C. 2044년 + 10번의 환희의 해 = B.C. 1554년
  6. B.C. 1554년 + 10번의 환희의 해 = B.C. 1064년
  7. B.C. 1064년 + 10번의 환희의 해 = B.C. 574년
  8. B.C. 574년 + 10번의 환희의 해 = B.C. 84년
  9. B.C. 84년 + 10번의 환희의 해 = A.D. 406년
  10. A.D. 406년 + 10번의 환희의 해 = A.D. 896년
  11. A.D. 896년 + 10번의 환희의 해 = A.D. 1386년
  12. A.D. 1386년 + 10번의 환희의 해 = A.D. 1876년
  13. A.D. 1876년 + 10번의 환희의 해 = A.D. 2366년

2) B.C. 1451년에서 시작한다면 다음과 같다 (여호수아가 약속의 땅에 들어간 해)

  1. B.C. 1451년 + 10번의 환희의 해 = B.C. 961년
  2. B.C. 961년 + 10번의 환희의 해 = B.C. 471년
  3. B.C. 471년 + 10번의 환희의 해 = A.D. 19년
  4. A.D. 19년 + 10번의 환희의 해 = A.D. 509년
  5. A.D 509년 + 10번의 환희의 해 = A.D. 999년
  6. A.D 999년 + 10번의 환희의 해 = A.D. 1489년
  7. A.D 1489년 + 10번의 환희의 해 = A.D. 1979년
  8. A.D 1979년 + 10번의 환희의 해 = A.D. 2469년

3) A.D 1979년 + 1 환희의 해 = 2028년

천 년이 하루인 단위 주기

  1. B.C. 4004년 - B.C. 3004년
  2. B.C. 3004년 - B.C. 2004년
  3. B.C. 2004년 - B.C. 1004년
  4. B.C. 1004년 - B.C. 4년 (그리스도의 탄생)
  5. B.C. 4년 - A.D. 996년
  6. A.D. 996년 - A.D. 1996년 (재림)
  7. A.D. 1996년 - A.D. 2996년 (천년왕국)

클라렌스 라킨 (Clarence Larkin)은 그리스도의 탄생을 B.C.4년으로 보기에 재림을 1996년으로 본다. "만일 우리의 달력이 정확하고," 또한 만일 하나님께서 다니엘 9:26-27 (욥 3:6 참조)에서 하셨던 것처럼 "어떤 해도 건너뛰지 않으신다면" 이것이 재림의 시기가 될 것이다. 만일

  • B.C. 3년에 태어나셨다면 재림은 1997년이 될 것이고,
  • B.C. 2년에 태어나셨다면 1998년이 될 것이며,
  • B.C. 1년에 태어나셨다면 1999년이 될 것이다. (그러나 지금은 2021년이다.)

    하나님께서 이방인들을 다루시는 2,500년

    "이방인의 때" = 2,500년 (레 26:18-28, 눅 21:24, 신 28:48-68, 계 11:2; 12:6, 14 참조)이 기간은 느부캇네살이 유다를 사로잡아 갔을 때부터 시작되었다.

    그때가 B.C. 606년이었다면 (단 1:1-2) + 2,520년 = A.D. 1914년

    그것이 B.C. 588년이었다면 (왕하 25:3-4) + 2,520년 = A.D. 1932년

    그것이 B.C. 536년이었다면 (스 1:1-4; 3:1-4) + 2,520년 = A.D. 1984년

    "이방인들의 날"이 끝나는 것과 관련하여 중요한 날짜들 : A.D. 1917년 - 벨푸어 선언 (Balfour Declaration), A.D. 1948년 - 이스라엘 민족이 국가가 됨, A.D. 1967년 - 6일 전쟁.
